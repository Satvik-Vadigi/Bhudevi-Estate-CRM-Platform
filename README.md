# 🏡 Bhudevi Estate CRM Platform

A production-ready full-stack Real Estate CRM platform built using Next.js, TypeScript, Prisma, PostgreSQL, and Vercel.

## 🚀 Live Demo

🔗 https://your-vercel-link.vercel.app

## 📌 Overview

Bhudevi Estate CRM Platform helps real estate businesses manage:

- Property listings
- Customer inquiries
- Lead pipeline
- Secure admin dashboard
- Real-time database operations

---

## ✨ Features

### Public Website

- Landing page
- Listings page

- Secure login

### ⚙️ Installation

## CMD

```bash
git clone https://github.com/Satvik-Vadigi/Bhudevi-Estate-CRM-Platform.git
cd Bhudevi-Estate-CRM-Platform
npm install
```

## Create .env

```dash
DATABASE_URL=your_postgresql_url
```

## Run project:

``` bash
npx prisma generate
npx prisma db push
npm run dev
```
